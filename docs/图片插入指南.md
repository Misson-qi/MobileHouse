## ⚠️ 注意事项

1. **路径正确性**
   - 所有图片路径都是相对路径
   - 从 `docs/` 目录引用 `assets/`
   - 格式：`../assets/images/chapterXX/xxx.jpg`

2. **图片位置**
   - 建议在小节标题下方插入
   - 每个小节不超过2-3张图片
   - 避免连续多张图片，中间要有文字说明

3. **图片说明**
   - 在图片下方加斜体说明文字（可选）
   - 格式：`*图片说明文字*`

4. **图片大小**
   - Markdown 会自动适配图片大小
   - 如需调整，可使用 HTML 标签：
     ```html
     <img src="../assets/images/chapter05/xxx.jpg" width="600" alt="描述">
     ```
